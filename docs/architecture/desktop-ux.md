# Desktop UX Contract

## Taskbar Zones
- Left: launcher entrypoint + global search
- Center: pinned and running app tasks
- Right: system tray, notifications, clock, power

## Launcher
- Search-first interaction for apps, settings, and actions.
- Supports pinned favorites and recent commands.

## System Apps (baseline)
- settings
- files
- terminal
- software-center
- system-monitor
- logs

## Session Rules
- Shell launches only after compositor and policy services are healthy.
- Critical tray indicators must always remain available.

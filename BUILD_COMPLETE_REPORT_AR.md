## 🔧 كيفية الاستخدام

### 1. **إنشاء ISO فعلية على Linux:**

```bash
# باستخدام mkisofs
mkisofs -R -J -V CARROTOS_1.0 \
  -b boot/bootloader \
  -o carrotos-1.0-x86_64.iso \
  /path/to/iso_staging

# أو باستخدام xorriso (الأفضل)
xorriso -as mkisofs -iso9660-level 3 \
  -V CARROTOS_1.0 \
  -o carrotos-1.0-x86_64.iso \
  /path/to/iso_staging
```

### 2. **كتابة على USB:**

```bash
# على Linux
sudo dd if=carrotos-1.0-x86_64.iso of=/dev/sdX bs=4M
sudo sync

# على Windows (استخدام Rufus أو GNOME Disks)
```

### 3. **التشغيل الافتراضي:**

```bash
# باستخدام QEMU
qemu-system-x86_64 -cdrom carrotos-1.0-x86_64.iso -m 2G

# أو استخدام VirtualBox أو VMware
```

```

│      Applications Layer              │  ← 7 تطبيقات متكاملة

├─────────────────────────────────────┤

│   Desktop Environment Layer         │  ← shell.cpp (واجهة المستخدم)
│   (Window Manager, UI Framework)   │
├─────────────────────────────────────┤
│    Kernel & Init Layer              │  ← kernel.o + init.o
│  (Process Management, IPC)          │
├─────────────────────────────────────┤
│     Bootloader Layer                │  ← bootloader.o
│  (Hardware initialization)          │
├─────────────────────────────────────┤
│     Hardware Layer                  │  ← CPU, Memory, Devices
└─────────────────────────────────────┘
```
